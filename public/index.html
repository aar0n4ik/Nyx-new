Capabilities

Everything runs on the edge

A complete private-AI stack — reasoning, retrieval, autonomous OS actions and security — with nothing leaving the device.

🧠

Multi-agent reasoner

Router → retriever → reasoner pipeline that grounds answers in your local notes.

🌍

6-language assistant

Auto-detects EN/RU/UK/ES/DE/FR and always replies in the user’s language.

🤖

Autonomous PC control

Diagnoses an issue, then writes and runs the fix (PowerShell/Bash) itself — time, language, updates, settings — with one-click confirm and Windows UAC.

📜

Proof-of-Local-Inference

Signed, verifiable hash-chain proving every answer was computed locally.

🛡️

NetGuard + injection firewall

Blocks egress and scrubs prompt-injection from retrieved context before the model.

💸

USD₮ fee accounting

Tracks per-inference fees in USD₮ on-device; full micro-settlement through the Tether WDK is on the roadmap.

Trust, proven

Why a judge can believe it

Every privacy claim ships with machine-checkable evidence — not marketing.

✔
Signed inference log

Ed25519 hash-chain; verify with npm run verify.

✔
Zero egress evidence

NetGuard logs every socket and blocks all egress except the allowlist (loopback + Bitfinex); unapproved connections = 0.

✔
Injection firewall

Untrusted context is scanned + sanitized before the model.

✔
Hardened security layer

All local API keys and private variables are sealed with high-grade, device-native encryption — secrets never touch the cloud.

✔
Proactive trading protection

Before any Bitfinex order executes, Nyx runs interactive dual-stage security checks, explicit confirmations and automatic safety overrides — guaranteeing full human control.

✔
Attestation

Device + model hashes captured for each session.
